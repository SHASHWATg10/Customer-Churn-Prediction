# Customer Churn Prediction

A professional Machine Learning web application built with Streamlit.

## Features

- Customer Input Form
- Decision Tree Pipeline
- Prediction Probability
- Download Prediction
- Modern Dashboard
- Error Handling

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
